CX INTELLIGENCE SUITE - PACKAGE AND TECHNOLOGY INVENTORY
Prepared for IT review
Version scope: IntelligenceSuite-WebVersion and equivalent desktop packaging approach
Last updated: 2026-07-01

1. APPLICATION SUMMARY

Tool name:
Kestrel Analysis Suite / CX Intelligence Suite

Primary purpose:
Analyze CSAT and NPS survey workbooks, generate performance dashboards, statistical results, role-based insight lenses, sentiment output, theme classification, ACPT classification, resolution status, and export-ready outputs.

Deployment options:
1. Web version hosted as a Python web service.
2. Desktop version packaged as a standalone local app with bundled runtime/dependencies.


2. RUNTIME STACK

Web runtime:
Python 3.11.9

Web server:
Python standard library HTTP server pattern using ThreadingHTTPServer/BaseHTTPRequestHandler in backend/server.py

Frontend:
Static HTML, CSS, and JavaScript.
No Node.js build step is required for the current WebVersion.
No package.json is present in the WebVersion.

Render deployment:
Build command:
pip install --upgrade pip && pip install -r requirements.txt

Start command:
python backend/server.py

Health check:
/healthz

9. DESKTOP VERSION DEPENDENCY POSITION

For the desktop version, the same Python package set should be bundled with the application or included in the packaged runtime. Users should not need to install Python manually if the desktop package includes:

1. Python runtime.
2. Required packages from requirements.txt.
3. Static frontend assets.
4. Backend Python server.
5. Required model artifacts.
6. Launcher EXE.


10. WEB VERSION DEPENDENCY POSITION

For the web version, Render or another hosting platform installs Python packages from requirements.txt during deployment. The hosted service then serves frontend files and backend APIs from the same application.

Recommended model storage approach for web:
1. Keep source code in Git.
2. Avoid committing newly trained large model files unless approved by IT.
3. Use persistent disk or external storage for trained model artifacts.
