What is a Base File?

Your Base File is the main CSAT survey export.

It usually contains one row per customer response, with fields such as CSAT score, satisfaction category, comment, feedback date, agent, manager, queue, channel, wave, or tenure. If some of those fields are missing, do not worry. After this upload, I will ask whether you want to add a lookup file and I can join both files using a unique key.

Accepted formats: .xlsx or .xls. Keep the original column headers in the first row.

How to read
  1. Start with the final decision. Favorable means the signal is healthy, Watch means monitor closely, and Action means review or intervene.
  2. Open each question. Review the assessment cards, calculation logic, and evidence table supporting the answer.
  3. Inspect the source evidence. Double-click cards or rows to see the exact data points, examples, and artifacts behind the conclusion.
  4. Check the comparison context. Confirm the selected current period, previous period, target, and week window before interpreting movement.
  5. Validate confidence. Review response volume, data completeness, and comparable periods before treating a signal as conclusive.
  6. Turn the signal into action. Record the owner, due date, expected outcome, and follow-up review for any item requiring attention.
  7. Find where the signal is concentrated. Compare agents, managers, themes, channels, and other available groups to identify where attention will have the greatest effect.
  8. Confirm the outcome. Revisit the lens after the follow-up period and verify whether the action improved the result without weakening another measure.
What is COPC?

COPC is a customer-experience operations framework for consistent performance management. It connects clearly defined measures with reliable evidence, disciplined review, and accountable improvement actions.

What COPC-aligned means in this tool
  • Defined measuresEach lens explains the metric, classification logic, target, and period used.
  • Comparable evidenceCurrent and previous periods are assessed using consistent calculation rules.
  • Reliable conclusionsVolume, completeness, confidence, and supporting examples are shown for validation.
  • Action-oriented reviewSignals identify what is favorable, what needs watching, and where action is required.
Important This is an internal COPC assessment-readiness view. It supports evidence-based operational reviews but does not represent COPC certification.
