Batch Loss Curve

Rolling Loss Trend

Lower and smoother is better. This averages recent batches so the overall learning direction is easier to read.

Confusion Heatmap

Rows are actual labels and columns are predicted labels. Strong diagonal values mean the model is separating classes well.
